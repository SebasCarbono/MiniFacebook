# MiniFacebook

En este proyecto usamos el coeficiente de Jaccard para crear recomendaciones optimas de amistad entre dos personas. Para entenderlo de manera sencilla podemos pensarlo como: 
"Si Sebastían y Daniela comparten X cantidad de amigos en común pero no son amigos entre sí, ¿que tan probable es que se cree una amistad con base a la similitud de su grupo de amigos?"
